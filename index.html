3 Detection Modes

ScreenEagle offers 3 detection modes to fit your needs

Stop Shot

Triggers an alert when motion stops for a specified duration

  • Set duration in days, hours, minutes, and seconds
  • Perfect for monitoring ongoing activities
Example: Alert when a video stream freezes, game character stops moving, or a machine stops running

Start Shot

Triggers an alert when new motion is detected

  • Instant alert when movement begins
  • Ideal for security monitoring
Example: Alert when someone enters the frame, an application window changes, or a download starts

Cycling Shot

Captures screenshots at regular intervals

  • Scheduled Mode: Set start/end date with capture interval
  • Immediate Mode: Start now with set duration
  • Great for time-lapse monitoring
Example: Capture CCTV every hour, monitor construction site daily, or track progress on tasks
